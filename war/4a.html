Vragen

1:
Een data feed is een mechanisme om als gebruiker nieuwe informatie te krijgen en is veelal ook real-time.
De term kan ook gebruikt worden op het world wide web. Dit kunnen documenten zijn om updates van o.a. websites binnen te krijgen.

2:
Het verschil zit in de notatie van de documenten. XML is een algemene opmaak voor documenten die er voor zorgd dat mensen en machines de gegevena kan lezen.
RSS is een standaard die gebaseerd is op XML en een aantal verplichte velden kent.
JSON is een formaat dat veel compacter en niet, zoals XML, een validatie mechanisme kent.
Omdat JSON compacter is heeft het minder overhead dan XML.
